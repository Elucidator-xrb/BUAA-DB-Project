## 0 命名规则
1.命名的方式为 当前页面/[对象]/功能。如login/login表示login页面的login功能  
2.使用HTTP自身的方法区分增删改查资源。  
 GET：查询，POST：新增，PUT：更新，DELETE：删除  
（这个我之前理解错了，不是用这些去表示请求的方法的）。  
如user/GET/users 表示在用户页面获取所有用户的信息
